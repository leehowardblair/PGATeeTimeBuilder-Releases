# PGA Tee Time Builder v1.1.3

## Release Date
<i>July 1, 2026</i>

## ✨ Highlights
- Cleaner post-build completion window with compact action buttons.
- macOS output choice: Excel, Numbers, or Both.
- Expanded Issue Report ZIP diagnostics for easier support troubleshooting.

## 📝 Notes
- Excel remains the default output format.
- Native Numbers export requires Apple Numbers.
- macOS may ask once for permission to allow PGA Tee Time Builder to control Numbers when using Numbers or Both.
- You may have to have numbers OPEN during first attempt at generating a numbers version.

## 🐞 Fixes
- Signed and notarized macOS Dev by Lee Blair TV
- macOS v1.1.3 should install on Intel or Apple Silicon chipsets

New users can simply download and run:

WINDOWS
